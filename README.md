# COMMERCIA

Sistema web para gerenciamento de vendas de itens, onde vendedores podem cadastrar produtos e usuários podem visualizar e realizar compras com controle automático de estoque.

---

## Problemática

Muitas pessoas enfrentam dificuldade para organizar e divulgar produtos para venda em plataformas simples e acessíveis. Além disso, pequenos vendedores nem sempre possuem ferramentas para gerenciar estoque e vendas de forma eficiente.

---

## Proposta

O sistema visa criar uma plataforma de vendas onde:

- Vendedores podem cadastrar, editar e excluir produtos  
- Usuários podem visualizar os itens disponíveis  
- Usuários podem realizar compras  
- O sistema atualiza automaticamente o estoque após cada venda
